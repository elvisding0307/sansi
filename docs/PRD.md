# SanSi PRD

## 一、产品概述

### 1.1 产品定位

AI 驱动的美股专业研究工具。输入股票代码，生成券商研报级别的深度分析报告。

SanSi（三思）扮演专业金融分析师的角色——客观、数据驱动，不做说教，不设行为干预。提供充分信息，让用户自己做判断。

### 1.2 目标用户

- 有基础的财务知识，能读懂财报指标
- 希望在做决策前获得系统性的分析参考
- 自主决策，不需要手把手引导

### 1.3 市场范围

仅覆盖美股（NYSE / NASDAQ）。

---

## 二、核心功能

### 2.1 报告生成

输入股票代码和公司名称，自动生成一份完整的个股深度研报。

报告包含十个模块（详见 [equity_research_template.md](./equity_research_template.md)）：

| # | 模块 | 说明 |
|---|------|------|
| 1 | 投资概要 | 评级、目标价、核心论点、财务快照 |
| 2 | 投资逻辑与风险 | 多支柱投资逻辑 + 风险对称分析 |
| 3 | 行业分析 | 市场规模、竞争格局、波特五力、趋势 |
| 4 | 公司分析 | 业务模式、管理层、业务拆分、护城河、成长驱动 |
| 5 | 财务分析 | 收入、盈利、杜邦、现金流、偿债能力 |
| 6 | 盈利预测 | 核心假设 + 预测损益表 + 三情景 |
| 7 | 估值分析 | DCF + 可比公司 + 历史区间 + Football Field |
| 8 | 技术面分析 | 趋势、关键价位、动量指标 |
| 9 | 后续关注信号 | 关键跟踪变量与判断标准 |
| 10 | 风险提示 | 宏观/行业/经营/估值四维风险 |

### 2.2 数据获取

- 主数据源：FMP API（Financial Modeling Prep）
- 备用数据源：yfinance（自动降级）
- LLM：DeepSeek API（兼容 OpenAI SDK）

### 2.3 Web 管理后台

- 用户认证（本地账号密码登录）
- 报告任务提交与进度追踪
- 历史报告查看与下载（HTML / PDF）
- 管理后台（用户管理、请求统计）

---

## 三、数据流

```
用户输入（代码 + 公司名）
  │
  ├─ [并行] FMP 数据获取 ──→ yfinance 降级
  │
  ├─ 财务数据处理（指标提取、预测、杜邦分析）
  │
  ├─ LLM 文本生成（综合分析，交叉验证）
  │
  ├─ 图表生成（matplotlib）
  │
  └─ 报告渲染（HTML → PDF）
```

## 四、技术栈

| 层 | 技术 |

## 五、配置

所有配置通过环境变量管理（`.env` 文件）：

| 变量 | 说明 | 必需 |
|------|------|------|
| `FMP_API_KEY` | Financial Modeling Prep API Key | 否（降级 yfinance） |
| `DEEPSEEK_API_KEY` | DeepSeek API Key | 是 |
| `DEEPSEEK_BASE_URL` | DeepSeek 接口地址 | 否（默认官方地址） |
| `DEEPSEEK_MODEL` | 模型选择 | 否（默认 deepseek-chat） |
| `ADANOS_API_KEY` | 散户情绪数据（可选） | 否 |

---

## 六、非功能需求

- 报告生成时间：单只股票 < 5 分钟
- 支持并发任务处理
- 任务日志持久化（文件 + 数据库）
- 密码哈希存储

---

## 七、不做什么

- 不提供买入/卖出建议
- 不做行为干预或情绪检测
- 不支持 A 股（仅美股）
- 不做实时行情推送
- 不做投资组合管理
