## 模块二：投资逻辑与风险

### 2.1 投资逻辑（3 个以上支柱）

每个支柱三段式展开：

**第一段：市场机会量化**
- 赛道规模、增速、驱动因素
- 公司在其中的位置和份额
- 用具体数字，不用模糊表述

**第二段：公司凭什么赢**
- 竞争优势（品牌、技术、渠道、规模、牌照、成本结构）
- 护城河宽度与可持续性
- 与竞争对手的差异化

**第三段：财务影响**
- 这项优势能带来多少增量收入 / 利润
- 何时兑现（时间线）
- 对估值的影响

### 2.2 风险分析

- 触发概率：低 / 中 / 高
- 影响程度：有限 / 中等 / 重大
- 缓解因素

## 模块十：风险提示

### 10.1 风险清单

每条风险格式：**风险名称** + 具体描述 + 可能影响 + 概率判断。

#### 宏观风险
- 宏观经济复苏不及预期 → 影响行业整体需求
- 利率持续上行 → 压制估值倍数
- 汇率大幅波动 → 影响海外收入折算
- 地缘政治风险 → 影响供应链 / 市场准入

#### 行业风险
- 竞争格局恶化 → 价格战侵蚀利润
- 技术路线替代 → 现有产品/服务被淘汰
- 政策监管趋严 → 合规成本上升 / 业务受限
- 下游需求结构性萎缩 → 行业天花板下行

#### 经营风险
- 客户集中度过高 → 大客户流失冲击收入
- 原材料价格大幅波动 → 毛利率不稳定
- 产能扩张不及预期 → 增长逻辑被打断
- 核心人才流失 → 执行力下降
- 并购整合失败 → 商誉减值风险

#### 估值风险
- 当前估值已充分反映乐观预期 → 预期落空导致杀估值
